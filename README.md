# STM32 多通道环境采集与阈值报警系统
基于 STM32F103C8T6 标准库开发，实现多路ADC采集＋DMA数据转运、OLED显示、按键交互、声光报警、串口数据上传功能。

## 开发环境
- Keil MDK5 + STM32F10x 标准库
- 串口调试助手（波特率 9600）
- 下载器：ST-Link / DAPLink

## 硬件接线
### ADC 模拟采集
- 电位器：PA1
- 热敏电阻：PA2
- 光敏电阻：PA3
### 按键
- 独立按键：PA0（上拉输入，另一端接GND）
### 执行器
- 蜂鸣器：PB12
- LED指示灯：PB13
### OLED I2C
- SCL：PB8
- SDA：PB9
- VCC：3.3V，GND：GND

## 项目功能
1. 三路ADC实时采集电位器、温度、光照数据
2. 独立按键切换OLED分页菜单
3. 电位器动态设置温度、光照报警阈值
4. 数值超标自动触发蜂鸣器+LED声光报警
5. 串口实时向上位机打印原始ADC数据
6. 模块化分层架构，驱动层与业务逻辑分离

## 使用步骤
1. 按照上方引脚完成硬件接线
2. 使用 Keil 打开 `.uvprojx` 工程文件
3. 编译无误后通过 ST-Link 下载程序
4. 打开串口助手即可查看实时数据
