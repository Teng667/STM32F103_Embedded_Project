项目一：多通道环境采集与阈值报警系统
项目介绍
本项目为 STM32 入门级综合练习项目，整合常用外设驱动与多通道模拟数据采集功能，完整掌握 GPIO、ADC、I2C、USART 等底层外设配置与使用，采用模块化分层架构，为后续复杂传感器项目提供稳定工程基础。
硬件设备
STM32F103C8T6 主控最小系统板
0.96 寸 I2C 接口 OLED 显示屏
电位器、热敏电阻、光敏电阻（模拟环境传感器）
独立功能按键、LED 状态指示灯、有源蜂鸣器
硬件接线
ADC 模拟采集
电位器：PA1
热敏电阻：PA2
光敏电阻：PA3
按键
独立按键：PA0（上拉输入，另一端接 GND）
执行器
蜂鸣器：PB12
LED 指示灯：PB13
OLED I2C
SCL：PB8
SDA：PB9
VCC：3.3V，GND：GND
实现功能
三路 ADC 结合 DMA 实时采集电位器、温度、光照数据
独立按键切换 OLED 分页菜单
电位器动态设置温度、光照报警阈值
数值超标自动触发蜂鸣器 + LED 声光报警
串口实时向上位机打印原始 ADC 数据
模块化分层架构，驱动层与业务逻辑分离
核心技术点
ADC 多通道扫描模式 + DMA 自动数据转运
GPIO 输入输出模式配置与外设联动控制
I2C 通信时序协议与 OLED 屏幕驱动实现
串口波特率配置、数据格式化收发与调试打印
嵌入式工程模块化架构与代码规范
开发环境
Keil MDK5 + STM32F10x 标准库
串口调试助手（波特率 9600）
下载器：ST‑Link / DAPLink
使用说明
按照上方引脚完成硬件接线
使用 Keil 打开 .uvprojx 工程文件
编译无误后通过 ST‑Link 下载程序
打开串口助手即可查看实时数据
项目二：MPU6050 姿态检测与 SG90 舵机随动系统
项目介绍
在项目一工程框架基础上扩展实现六轴姿态解算与闭环随动控制系统。通过 MPU6050 传感器获取加速度与陀螺仪原始数据，采用互补滤波算法完成数据融合，解算俯仰角、滚转角；结合一阶低通滤波、PD 闭环控制算法，实现舵机跟随姿态顺滑转动，有效抑制抖动与超调，是典型嵌入式传感器 + 算法 + 执行器综合实战项目。
硬件设备
STM32F103C8T6 主控最小系统板
MPU6050 六轴姿态传感器（I2C 通信）
SG90 微型舵机（TIM PWM 驱动）
0.96 寸 OLED 显示屏、串口通信模块
硬件接线
MPU6050 姿态传感器
VCC → 3.3V
GND → GND
SDA → PB11
SCL → PB10
SG90 舵机
控制信号线 → PA0（TIM2_CH1）
VCC → 5V
GND → GND
实现功能
通过 I2C 模拟时序读写 MPU6050，获取六轴加速度、陀螺仪原始数据
互补滤波算法完成传感器数据融合，解算 Pitch 俯仰角、Roll 滚转角
一阶低通滤波平滑姿态数据
PD 闭环控制算法优化舵机响应速度与转动稳定性
滚转角实时映射为舵机角度，实现倾斜角度越大偏转越大的随动效果
OLED 实时显示当前姿态角，串口定时打印角度与调试信息
工程结构完全复用项目一驱动，外设统一管理，耦合度低
核心技术点
I2C 模拟时序通信实现传感器寄存器读写
六轴姿态解算原理、互补滤波数据融合算法实现
TIM 定时器 PWM 输出配置与舵机角度驱动
PD 闭环控制原理、参数整定与嵌入式算法实现
多外设并行调度、模块化分层驱动开发
开发环境
主控芯片：STM32F103C8T6
开发库：STM32 标准库
编译软件：Keil5 MDK
系统主频：72MHz
调试工具：串口助手、OLED 屏幕实时显示
使用说明
按接线表连接 MPU6050、SG90 舵机及原有外设
Keil 编译下载程序，上电后舵机自动回中
倾斜开发板，观察舵机跟随滚转角同步转动
OLED 实时显示俯仰角、滚转角，串口持续输出姿态数据
