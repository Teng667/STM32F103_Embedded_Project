项目一：多通道环境采集与阈值报警系统
##项目介绍
本项目为 STM32 入门级综合练习项目，整合常用外设驱动与多通道模拟数据采集功能，完整掌握 GPIO、ADC、I2C、USART 等底层外设配置与使用，采用模块化分层架构，为后续复杂传感器项目提供稳定工程基础。

##硬件设备
STM32F103C8T6 主控最小系统板
0.96 寸 I2C 接口 OLED 显示屏
电位器、热敏电阻、光敏电阻（模拟环境传感器）
独立功能按键、LED 状态指示灯、有源蜂鸣器

##硬件接线
ADC 模拟采集通道
电位器（阈值调节）：PA1
热敏电阻（温度采集）：PA2
光敏电阻（光照采集）：PA3
按键模块
独立按键：PA0（内部上拉输入，另一端接 GND）
声光报警模块
有源蜂鸣器：PB12
LED 状态指示灯：PB13
OLED I2C 显示屏
I2C_SCL：PB8
I2C_SDA：PB9
供电：3.3V，GND

##实现功能
GPIO 输入输出控制，实现按键扫描、LED 指示、蜂鸣器声光报警
ADC 多通道循环采集，完成电压、温度、光照模拟量读取与换算
I2C 通信驱动 OLED 分页显示实时数据与报警阈值
USART1 串口向上位机格式化打印原始数据与状态信息
按键切换 OLED 显示页面，电位器动态修改报警阈值
数值超标自动触发声光报警，异常解除后自动恢复
模块化分层代码设计，驱动层与业务逻辑分离，便于移植与二次开发

##核心技术点
GPIO 推挽输出、上拉输入模式配置与电平控制
ADC 连续转换、多通道扫描模式模数转换原理
I2C 通信时序协议与 OLED 屏幕驱动实现
串口波特率配置、数据格式化收发与调试打印
嵌入式工程模块化架构与代码规范
简单阈值判断与外设联动控制逻辑

##开发环境
编译软件：Keil5 MDK
固件库：STM32F10x 标准库
系统主频：72MHz
下载调试：ST‑Link
上位机工具：串口调试助手（波特率 9600）
##使用说明
按照硬件接线表完成全部线路连接，确认电源无短路
使用 Keil 打开对应工程文件，编译工程无报错后下载程序
打开串口调试助手，配置波特率 9600，查看实时采集数据
短按按键切换 OLED 显示页面，旋转电位器调节报警阈值
遮挡光敏电阻或加热热敏电阻，观察声光报警效果

项目二：MPU6050 姿态检测与 SG90 舵机随动系统
##项目介绍
在项目一工程框架基础上扩展实现六轴姿态解算与闭环随动控制系统。通过 MPU6050 传感器获取加速度与陀螺仪原始数据，采用互补滤波算法完成数据融合，解算俯仰角、滚转角；结合一阶低通滤波、死区处理、PD 闭环控制算法，实现舵机跟随姿态顺滑转动，有效抑制抖动与超调，是典型嵌入式传感器 + 算法 + 执行器综合实战项目。

##硬件设备
STM32F103C8T6 主控最小系统板
MPU6050 六轴姿态传感器（I2C 通信）
SG90  微型舵机（TIM PWM 驱动）
0.96 寸 OLED 显示屏、串口通信模块

##硬件接线
#MPU6050 姿态传感器
VCC → 3.3V
GND → GND
SDA → PB11
SCL → PB10
SG90 舵机
控制信号线 → PA0（TIM2_CH1）
VCC → 5V
GND → GND
OLED、串口与项目一保持一致

##实现功能
通过 I2C 模拟时序读写 MPU6050，获取六轴加速度、陀螺仪原始数据
互补滤波算法完成传感器数据融合，解算 Pitch 俯仰角、Roll 滚转角
一阶低通滤波平滑姿态数据
PD 闭环控制算法优化舵机响应速度与转动稳定性
滚转角实时映射为舵机角度，实现倾斜角度越大偏转越大的随动效果
OLED 实时显示当前姿态角，串口定时打印角度与调试信息
工程结构完全复用项目一驱动，外设统一管理，耦合度低

##核心技术点
I2C 模拟时序通信实现传感器寄存器读写
六轴姿态解算原理、互补滤波数据融合算法实现
TIM 定时器 PWM 输出配置与舵机角度驱动
PD 闭环控制原理、参数整定与嵌入式算法实现
多外设并行调度、模块化分层驱动开发

##开发环境
主控芯片：STM32F103C8T6
开发库：STM32F10X 标准库
编译软件：Keil5 MDK
系统主频：72MHz
调试工具：串口助手、OLED 屏幕实时显示

##使用说明
按接线表连接 MPU6050、SG90 舵机及原有外设
Keil 编译下载程序，上电后舵机自动回中
倾斜开发板，观察舵机跟随滚转角同步转动
OLED 实时显示俯仰角、滚转角，串口持续输出姿态数据
